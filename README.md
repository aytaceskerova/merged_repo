# Merged Database Repository

This repository contains merged SQL files from task1.sql and task2.sql.

## Contents

- `merged_database.sql` - Combined database schema and queries

## Database Schema

### Tables

1. **Employees** - Employee information with department and salary data
2. **customers** - Customer information with country data
3. **orders** - Order information linked to customers

## Queries

1. **Department Employee Count** - Finds departments with more than 5 employees
2. **Customer Statistics** - Complex query showing customer spending statistics with latest order information

## Usage

Run the SQL file in your database management system to create the tables, insert sample data, and execute the queries.
